Older (pre-HTML5) Versions

Prior to the HTML5 version (i.e. prior to version 4.0), Multigraph was an Adobe Flash application. The Flash version of Multigraph was/is used in much the same way as the current HTML5/JavaScript. The original www.multigraph.org web site explains how to use the Flash version, and all versions of it are available in our archive.
