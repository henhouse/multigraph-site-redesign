Multigraph

Graphs for the Web.

Multigraph is a program for creating 2-dimensional scientific data graphs on web pages. It can read data in a variety of formats, and allows you to customize the appearance of the graph. Written in JavaScript and compatible with modern web browsers, including mobile devices.

View on GitHub

Panning

Just click and drag with your mouse (or finger) to drag the graph up and down!

Zooming

Scroll in or scroll out using your mouse! Have hands? Just pinch to zoom!
